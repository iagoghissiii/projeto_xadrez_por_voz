# ♔ Xadrez por Voz

Plataforma web de xadrez controlada por voz, desenvolvida como **Projeto Integrador** com foco em acessibilidade para pessoas com limitações físicas.

## 🎯 Objetivo

Permitir que qualquer pessoa jogue xadrez utilizando apenas a **voz**, sem necessidade de mouse ou teclado.

## ✨ Funcionalidades

- 🎙️ **Controle por voz** — fale suas jogadas em português (ex: *"cavalo f três"*, *"e dois e quatro"*)
- 🔊 **Feedback em áudio** — o jogo narra os eventos em voz alta
- 🤖 **IA com Stockfish** — três níveis de dificuldade (Fácil, Médio, Difícil)
- 👥 **Dois modos** — Humano vs Humano ou Humano vs Computador
- 📊 **Análise pós-jogo** — revisão de precisão lance a lance
- 🌙 **Tema claro/escuro**
- 📱 **Responsivo** — funciona em desktop e mobile

## 🗣️ Comandos de Voz

| Fala | Ação |
|------|------|
| `"e dois e quatro"` | Peão de e2 para e4 |
| `"cavalo f três"` | Cavalo para f3 |
| `"dama captura h sete"` | Dama captura em h7 |
| `"roque menor"` | Roque curto |
| `"roque maior"` | Roque longo |
| `"status"` | Ouve o estado atual do jogo |

> 💡 Clique no botão **?** no cabeçalho para ver todos os comandos disponíveis.

## 🚀 Como usar

Acesse direto pelo navegador — não requer instalação.

> ⚠️ Use **Google Chrome** ou **Microsoft Edge** para o reconhecimento de voz funcionar corretamente.

## 🛠️ Tecnologias

- HTML5, CSS3, JavaScript (Vanilla)
- [Chess.js](https://github.com/jhlywa/chess.js) — lógica do jogo
- [Chessboard.js](https://chessboardjs.com/) — tabuleiro visual
- [Stockfish.js](https://github.com/nmrugg/stockfish.js) — engine de IA
- Web Speech API — reconhecimento e síntese de voz
